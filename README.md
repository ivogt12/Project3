# EduGrade

Grading made Easy!!

## About 
EduGrade is a MERN-Stack Application that allows teachers to create classes and upload grades. EduGrade aims to provide an intuitive site for both students and teachers.

## Technologies Used
-MongoDB, Express, React, Node.js
